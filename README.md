# NDVI-Analysis-Bhubaneswar
NDVI-based vegetation analysis using satellite data(Google Earth Engine)
#OVERVIEW

This project leverages satellite imagery to analyze vegetation dynamics over Bhubaneswar using NDVI (Normalized Difference Vegetation Index). It demonstrates how remote sensing can be used to track environmental changes over time.
#KEY HIGHLIGHTS

Cloud-filtered Sentinel-2 imagery
NDVI computation using NIR & Red bands
Time-series visualization of vegetation trends
Region-based analysis (ROI)

#Dataset
Copernicus Sentinel-2 Surface Reflectance
#Output Preview
NDVI trend chart across 2023
Mean NDVI image export
Spatial visualization of study area
